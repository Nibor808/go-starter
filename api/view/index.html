Go Starter

You can use Postman for making requests

Adding a user

  • make a POST request to localhost:5000/signupemail with form data { email: your-email-address}
  • check your email for the verification and click the link
  • then make a POST request to localhost:5000/signuppassword with form data { password:your-password }

Sign In/Out

Sign in with make a POST request to localhost:5000/signin with form data { email: your-email-address, password: your-password }

If signed in:

  • View all users at localhost:5000/users
  • View the current user at localhost:5000/user
  • create data with a POST request to localhost:5000/savedata - accepts raw json 
          {
        "values": {
            "key1": "val1"
        }
      }
  • update data with a POST request to localhost:5000/updatedata - accepts raw json 
          {
          "id": "5f1da5bbc2e2b24edd262d34", - id is taken from the json returned by savedata
          "values": {
              "key1": "val1",
              "key2": [
                  {
                      "name": "Mike DaRookie",
                      "age": "45"
                  }
              ],
              "key3": ["item 1", "item 2", "item 3"],
              "key4": [10, 20, 30, 40 , 50.1, 50.2],
              "key5": "Jane Doe"
          }
      }
  • read data with a GET request to localhost:5000/alldata

Sign out at localhost:5000/signout

Tokens and Sessions expire after 10 minutes - this can be changed in utils/db.go -> line:54 -> ttl := int32(600)

Changes to html require a manual restart
