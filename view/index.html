Go Starter

You can use Postman for making requests

Adding a user

  • make a POST request to localhost:5000/signupemail?email=your-email.com
  • check your email for the verification and click the link
  • then make a POST request to localhost:5000/signuppassword?password=your-password

Sign In/Out

Sign in at localhost:5000/signin?email=your-email.com&password=your-password

If signed in:

  • View all users at localhost:5000/users
  • View the current user at localhost:5000/user
  • create (localhost:5000/savedata), read (localhost:5000/alldata), and update (localhost:5000/updatedata) any json data via the DataController - accepts raw json
  • updating data can be done by adding the "id" from the json returned by savedata to the update json

Sign out at localhost:5000/signout

Tokens and Sessions expire after 10 minutes - this can be changed in utils/db.go -> line:54 -> ttl := int32(600)

Changes to html require a manual restart
