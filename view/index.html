Go Starter

You can use Postman for making requests

Adding a user

  • first make a POST request to localhost:5000/signupemail with form value for email
  • click link in verification email to remove the token and confirm the email given
  • then make A POST request to localhost:5000/signuppassword with a form value for password

Sign In/Out

Sign out at localhost:5000/signout

Sign in at localhost:5000/signin with form values for email and password

If signed in:

  • View all users at localhost:5000/users
  • View the current user at localhost:5000/user

Tokens and Sessions expire after 60 seconds - this can be changed in utils/db.go -> line:52 -> ttl := int32(60)
